# Cyber_shild
CyberShield is an advanced front-end educational platform designed to demonstrate password strength algorithms and simulate common adversarial attacks in a safe environment.
# CyberShield Password Strength Analyzer

A complete, production-quality, fully responsive web application that analyzes password strength in real time, estimates crack resistance, demonstrates common password attack methods safely for education, generates secure passwords, and teaches best password practices.

## How to Run

1. Download or clone this folder.
2. Open `index.html` in your web browser.

**OR**

For the best development experience, open this folder in VS Code and use the **Live Server** extension.

## Features List

- **Live Password Analyzer:** Get real-time feedback as you type, with a sophisticated scoring engine checking length, uppercase, lowercase, numbers, symbols, and predictable patterns.
- **Crack Time Estimator:** See how long it would realistically take to brute-force your password.
- **Educational Attack Simulations:** Safe, visual demonstrations of Brute Force and Dictionary attacks to educate on password vulnerabilities.
- **Secure Password Generator:** Generate strong, random passwords with customizable length and character sets. One-click copy functionality included.
- **Dark & Light Mode Toggle:** Fully themed user interface utilizing modern glassmorphism design.
- **Fully Responsive:** Adapts flawlessly to mobile, tablet, and desktop screens.
- **Educational Content:** Learn about the dangers of password reuse, dictionary attacks, 2FA, and password managers.

## Technologies Used

- **HTML5:** Semantic structuring and accessible tags.
- **Vanilla CSS3:** Custom properties (variables), media queries, flexbox/grid layouts, and advanced animations (glassmorphism, glowing neons).
- **Vanilla JavaScript (ES6+):** DOM manipulation, password validation logic, simulation timing execution, all without any external libraries or dependencies.
